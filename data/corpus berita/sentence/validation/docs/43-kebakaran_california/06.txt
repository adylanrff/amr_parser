Kebakaran liar terbesar di selatan California diperkirakan akan menjadi bencana api terbesar di negara bagian itu.

Kobaran api bergerak menuju barat laut Los Angeles pada Minggu, sementara pemadam kebakaran masih mencoba melindungi komunitas-komunitas di pesisir Santa Barbara. Bencana api Thomas belum reda hingga kini dan bahkan makin parah karena kencangnya pusaran angin Santa Ana.

Badan cuaca AS mengatakan fenomena Santa Ana akan membawa angin dengan kecepatan hingga 88 kilometer per jam.

Bencana api Thomas tercatat sebagai kebakaran terbesar ketiga dalam sejarah California, setelah menghanguskan hingga 269.000 hektar tanah dan merusak lebih dari 1.000 bangunan. Api itu menyebar ke area sebesar 10.000 hektare pada Sabtu.

Pemadam kebakaran California, Cal Fire, mengatakan baru 40 persen dari api Thomas sudah padam setelah pertama terjadi dua pekan lalu. Lebih dari 18.000 bangunan lainnya terancam rusak.

“Cuaca ini kondusif untuk bencana api karena kelembaban rendah mendukung besarnya api ke arah barat, timur dan utara,” kata Cal Fire.

Pemadam Cory Iverson meninggal Kamis lalu akibat luka bakar dan menghirup asap setelah sempat bertarung dengan api Thomas.

Kebun Binatang Santa Barbara tutup pada Minggu karena buruknya kualitas udara. Mereka juga mengirim burung California Condor yang terancam punah dan burun bangkai ke Los Angeles untuk menyelamatkan mereka.

Selebriti Oprah Winfrey yang memiliki rumah di Santa Barbara mengatakan dia “mendoakan kota itu. Angin kencang mengakibatkan petaka bagi pemadam kebakaran”.

“Semoga pemadam kebakaran dan keluarga mereka dilindungi. Mereka berjuang melawan api selama tiga pekan dan masih terus berjuang. Bayangkan kelelahan mereka! #ThomasFire,” tulis Oprah di Twitter.

Bencana Cedar Fire, akibat perbuatan manusia, adalah kebakaran terbesar dalam sejarah California dan terjadi pada 2003 lalu. Bencana itu melahap 273.000 hektare tanah di San Diego.
